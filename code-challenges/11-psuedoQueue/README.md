# PseudoQueue

Class that creates a queue out of two stacks

## Challenge

Create a new class called PseudoQueue that utilizes 2 stack objects and creates the methods enqueue and dequeue to add and remove items from the queue.

## Approach & Efficiency
* enqueue - O(1)
* dequeue - O(n)
